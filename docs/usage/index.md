## Overview

TrikuSec provides:

1. **Security Overview Dashboard** - At-a-glance view of compliance rates, warnings, OS distribution, and devices requiring attention
2. **Device Management** - View and manage all your servers and their compliance status
3. **Policy Management** - Define and apply security policies
4. **Report Analysis** - Detailed audit reports with recommendations
5. **Compliance Tracking** - Monitor compliance over time

## Quick Start

1. **Access the Dashboard** - Navigate to `https://localhost:8000` (or your configured domain)
2. **Review Security Overview** - Check compliance rates, warnings, and devices needing attention
3. **View Devices** - See all servers that have sent reports
4. **Check Compliance** - Review compliance status for each device
5. **Create Policies** - Define rules for your organization
